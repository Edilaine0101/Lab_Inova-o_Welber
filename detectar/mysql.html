Introdução ao MySQL

Entenda como funciona um dos bancos de dados mais populares do mundo

O que é o MySQL?

O MySQL é um sistema de gerenciamento de banco de dados relacional (RDBMS) amplamente utilizado em aplicações web. Ele é conhecido por sua eficiência, escalabilidade e confiabilidade.

A instalação do MySQL pode ser feita de várias formas. Em sistemas baseados em Linux, pode-se usar:

sudo apt update
sudo apt install mysql-server

Para Windows, pode-se baixar o MySQL Installer diretamente do site oficial.

Abaixo está um exemplo de criação de uma tabela e inserção de dados:

CREATE TABLE usuarios (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100),
    email VARCHAR(100)
);
INSERT INTO usuarios (nome, email) VALUES ('João Silva', 'joao@email.com');
